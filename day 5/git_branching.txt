1. git branch 
   => list all the branches and highlights the active/current branch

2. git branch <branch_name>
   => creates a new branch <branch_name>

3. git checkout <branch_name>
   => Jumps to the <branch_name> provided

4. git checkout -b <branch_name>
   => creates a new branch and jumps to that branch immediately

5. git merge <branch-name>

6. git status
   => Provides the current status of the repo

7. git clone <project-ssh-link>
   => Donloads the repo to local device

8. git reset
   => if need to delete commit 
   

8. git branch -D <branch_name>
